## Store Builder Web View

Below are **images and screenshots** of the Store Builder project, showcasing its **dynamic content management capabilities**.  

This project allows users to **edit and customize a store website** with full control over content, layout, and media. The full project details are available in its **individual README**.

### Key Features
- **Dynamic Content Management:** Add and update images, text, and media links.  
- **Custom Layouts:** Change image alignment/direction and show/hide sections or media.  
- **Styling Control:** Modify website colors and styles dynamically.  
- **Preview and Edit Modes:** Switch between **editing** and **preview** views to see changes live.  
- **Product Management:** Add products, timetables, and manage cart functionality.

### Technologies Used
- **Frontend:** React, Ant Design, Redux, Sass  
- **Backend:** Node.js  
- **Database:** PostgreSQL, SQLite  
- **Containerization:** Docker
