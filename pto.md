# Project-Details

## 1. PTO Application

**Objective:**  
This application is designed to manage employees, teams, and their Paid Time Off (PTO) requests efficiently. It provides separate views and functionalities for **admins** and **employees**.

**Key Features:**  
- CRUD operations for **teams**, **employees**, and **PTO requests**.  
- **Email notifications** using Twilio.  
- **Forgot and reset password** functionality.  
- Admin and employee dashboards with role-specific access.  

**Technologies Used:**  
- **Frontend:** React, Sass, TypeScript, Redux-Saga  
- **Backend:** Node.js, Express  
- **Database:** MongoDB
