## Store Builder Web View

Below is a **video demonstration** of the Store Builder project.  

This project allows you to **dynamically update a store website** using a web builder. The full project details are available in its **individual README**.  

**Key Features Demonstrated in the Video:**  
- Add and update **images and content** dynamically.  
- Change **image alignment/direction**.  
- **Hide or display sections** of the website.  
- Manage **timetables** and **products**.  
- Add products to the **cart** for purchase.  

**Technologies Used:**  
- **Frontend:** React, Ant Design, Redux, Sass
